## 📚 Smart Study Assistant (NER Chatbot)

This project is a simple NLP-based application that extracts named entities from unstructured documents like PDF and Word files.

It uses spaCy's Named Entity Recognition (NER) to identify entities such as person names, locations, dates, and organizations.

### 🔧 Features
- Upload PDF or DOCX files
- Extract and display named entities
- Supports multiple entity types (PERSON, GPE, DATE, etc.)
- Simple and interactive interface using Streamlit

### 🛠️ Tech Stack
- Python
- spaCy (NLP)
- Streamlit
- pdfplumber
- python-docx

### 📌 Use Case
Helps in quickly analyzing large documents by extracting important information in a structured format.
